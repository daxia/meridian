## 需求1：文章聚类接口 (待实现)

**需求编号**: 20260214007

**DD编号**: DD-ML-智能服务需求

### 1. 需求功能点

**1）聚类 API**:

- **描述**: 提供 HTTP 接口，接收一批文章的 Embeddings，返回聚类结果（哪些文章属于同一个 Topic）。
- **算法**: 建议使用 HDBSCAN 或类似无需预设 K 值的算法。
- **输入**: JSON List of `{id, embedding}`。
- **输出**: JSON List of `{cluster_id, article_ids}`。

### 2. 目标与约束

- **目标**: 准确将报道同一事件的多篇来源文章聚合。
- **约束**: Python 服务性能（需在数秒内完成数百篇文章的聚类）。
