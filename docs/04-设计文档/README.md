## 设计文档说明：

PRD文档转成设计文档，写到对应的设计文档中，

设计文档中，分别有下面的文档：

* DD-GUI-主页面需求.md
* DD-GUI-培养箱设置页面需求.md
* DD-GUI-手动调试页面需求.md
* DD-GUI-系统设置页面需求.md
* DD-GUI-模拟系统需求.md
* DD-GUI-实际系统需求.md
* DD-GUI-入口功能需求.md
* DD-APP-Camera模块需求.md
* DD-APP-Controller模块需求.md
* DD-APP-System模块需求.md
* DD-APP-Request接口模块需求.md
* DD-APP-Robot模块需求.md
* DD-TOOL-TOOL模块需求.md

涉及到哪个模块的修改需求，就需要在哪个模块的设计文档中，进行补充更新文档。

也就是根据PRD文档进行分析，，实际落地到对应的设计文档模块需求。
