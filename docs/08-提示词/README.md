# 08-提示词

本目录用于记录用户与 AI 协作过程中的关键提示词 (Prompts)，以便复盘与优化 AI 交互效果。

## 记录原则

仅在以下情况记录：
1.  用户明确提出“记录提示词”或“总结对话”。
2.  解决了极具挑战性的技术难题，值得复盘 Prompt 策略。

